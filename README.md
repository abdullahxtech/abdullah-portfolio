## ✨ Features

- 💡 Light/Dark Theme Toggle
- 📱 Fully Responsive Design
- 🧠 Animated UI with Framer Motion
- 🔍 Project Filtering & Search
- 📁 GitHub + Live Project Links
- 📬 Contact Form (MongoDB support coming soon)
